# React Practice 🚀

A collection of React projects built while learning the React ecosystem from the basics to advanced concepts.

Each folder is a standalone React application that focuses on a specific concept, hook, or feature.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/srv-23/reactPractice.git
```

Navigate to any project folder:

```bash
cd 05passwordGenerator
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

> **Note:** Every folder is an independent React project with its own setup and dependencies.
